### 💭 Cierre Reflexivo

1. **¿Qué sucede con los punteros cuando se inserta o elimina un nodo?**
   Los punteros del nodo cambian de referencia.
2. **¿Cómo afecta la posición de un nodo (inicio, medio, final) al tiempo de búsqueda?**
   Se tarda más dependiendo de la posición del nodo a encontrar.
3. **¿Qué ventajas tiene recorrer una lista enlazada frente a un arreglo?**
   Las listas enlazadas pueden crecer o decrecer dinámicamente.
4. **¿Cómo comprobar si una lista está vacía en Java?**
   Verificando que la cabeza (`head`) sea `null`.

### 💭 Cierre Reflexivo

* **¿Qué aprendiste sobre las referencias en una lista enlazada?**
  Son punteros que indican el siguiente nodo.
* **¿Qué pasa si se rompe un enlace?**
  El nodo anterior apunta directamente al nodo posterior al eliminado.
* **¿Cómo se relaciona con el código en Java?**
  Representa cómo las estructuras dinámicas manejan las referencias entre nodos.

## 📄 U2ACT2 - Especificación de la Pila

Incluye la descripción detallada del funcionamiento y métodos principales de una **pila (Stack)**:

* `push()` para insertar
* `pop()` para eliminar
* `peek()` para observar el elemento superior
* `isEmpty()` para verificar si la pila está vacía
